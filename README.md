# InvestHer: Women's Investment Research & Platform

## 🎯 Overview

**InvestHer** is a data-driven initiative designed to understand and empower women investors in India. This repository contains a comprehensive case study analyzing female investment behavior, supported by a Loveable-built interactive platform for women-focused financial guidance.

### What's Inside
- 📊 **Case Study PDF**: In-depth research on women's investment patterns, behavioral segmentation, and market opportunity analysis
- 🌐 **Interactive Platform**: Live Loveable application connecting female investors with personalized financial insights
